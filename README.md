# SKY MEAL — учёт кухни бортпитания

Node.js без внешних зависимостей. Хранилище: data/db.json.

## Запуск на VPS
```
git clone <repo> && cd ttalem
node server.js 3050            # проверка
pm2 start server.js --name ttalem -- 3050
```
nginx: проксировать поддомен (например ttalem.probuh.asia) на localhost:3050, Certbot как обычно.

## Роли
- Повар: выработка (взвешивание до/после), выпуск блюд за смену, приход сырья, остатки
- Технолог: техкарты (граммовки из Excel Альфии — ПРОВЕРИТЬ!), номенклатура, потери
- Директор: живая калькуляция, потери по поварам, остатки, журнал со сторно, экспорт 1С

## Логика себестоимости
Средневзвешенная. Приход задаёт цену сырья → выработка переносит стоимость
на полуфабрикат (цена/кг растёт на процент потерь) → выпуск списывает по
техкарте и приходует блюдо. Всё пересчитывается на лету.

## Обмен с 1С (следующий этап)
GET /api/1c/export?date=YYYY-MM-DD — готовые документы КомплектацияНоменклатуры
с проставленными суммами (авторасчёт себестоимости 1С НЕ используется).
Расширение для 1С БК будет забирать JSON и создавать документы непроведёнными,
бухгалтер проверяет и проводит. Поле code1c у номенклатуры — для сопоставления.
